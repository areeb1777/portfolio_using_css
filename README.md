## Installation

Follow these steps to install and run the project locally:

1. Clone the repository:

   ```sh
   git clone https://github.com/areeb1777/portfolio_using_tailwind.git
   cd portfolio_using_tailwind

   ```

2. Install dependencies:


sh
npm install

3. Run the development server:

sh
npm run dev
Open your browser and navigate to http://localhost:3000 to see the portfolio.

Usage
You can use this project as a starting point for your own portfolio. Customize the content in the src/app/components directory to reflect your own skills and projects.

Technology Stack
Next.js: The React framework for production.

Tailwind CSS: A utility-first CSS framework for styling.

TypeScript: JavaScript with syntax for types.

EmailJS: Email service for handling contact form submissions.

React Icons: Icons library for React.
